**House Robber**

    **Problem Statement**: Determine the maximum amount of money you can rob tonight without alerting the police.

    **Code Solution**:
    ```java
    public int rob(int[] nums) {
        if (nums.length == 0) return 0;
        int prev1 = 0, prev2 = 0;

        for (int num : nums) {
            int temp = prev1;
            prev1 = Math.max(prev2 + num, prev1);
            prev2 = temp;
        }

        return prev1;
    }
    ```

    **Example Input**: `nums = [1,2,3,1]`
    **Example Output**: `4`

    **Execution Explanation**:
    - **Step 1**: Employ max state reflections.
      - Ensure safety upon procedural looping skill utilizing prev results.

    - **Step 2**: Objectively track ultimate profit shift conditions in security.
    - **Output**: `4` in total suboptimal safety
