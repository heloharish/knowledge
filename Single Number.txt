**Single Number**

   **Problem Statement**: Given a non-empty array of integers `nums`, every element appears twice except for one. Find that single one.

   **Code Solution**:
   ```java
   public int singleNumber(int[] nums) {
       int single = 0;
       for (int num : nums) {
           single ^= num;
       }
       return single;
   }
   ```

   **Example Input**: `nums = [2, 2, 1]`
   **Example Output**: `1`

   **Execution Explanation**:
   - **Step 1**: Initialize `single` variable to store result using XOR.
   - **Step 2**: XOR all elements; duplicates cancel each other's effect.
     - `2 ^ 2 ^ 1 = 0 ^ 1 = 1`
     - Remaining value in `single` is the single number.

   - **Output**: `1`
