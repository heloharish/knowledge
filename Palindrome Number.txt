**Palindrome Number**

   **Problem Statement**: Given an integer `x`, return `true` if `x` is palindrome integer.

   **Code Solution**:
   ```java
   public boolean isPalindrome(int x) {
       if (x < 0 || (x % 10 == 0 && x != 0)) return false;

       int reversed = 0;
       while (x > reversed) {
           reversed = reversed * 10 + x % 10;
           x /= 10;
       }

       return x == reversed || x == reversed / 10;
   }
   ```

   **Example Input**: `x = 121`
   **Example Output**: `true`

   **Execution Explanation**:
   - **Step 1**: Special scenario disqualified, handle negative.
   - **Step 2**: Progress last half reversed evaluation.
     - Slowly reverse number.
     - Check if mirror sets verify true for full palindrome certification.

   - **Output**: `true` (it's palindromic)
